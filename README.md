# Housing Price Prediction (Machine Learning)

## Overview
This project focuses on predicting housing prices using machine learning techniques and analyzing key factors affecting property prices.

## Objectives
- Perform data cleaning and preprocessing
- Handle missing values and inconsistencies
- Build predictive models for price estimation
- Visualize insights using dashboards

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn, XGBoost
- Matplotlib, Seaborn
- Tableau (for dashboard visualization)

## Steps Performed

### 1. Data Preprocessing
- Handled missing values
- Data cleaning and transformation
- Feature engineering

### 2. Exploratory Data Analysis (EDA)
- Identified trends and correlations
- Feature importance analysis

### 3. Model Building
- Implemented regression models
- Used XGBoost for improved performance

### 4. Model Evaluation
- Achieved best R² score of 0.774

### 5. Visualization
- Developed interactive Tableau dashboards for insights

## Key Insights
- Data quality significantly impacts model performance
- Feature selection improves prediction accuracy
- Location and property features strongly influence pricing

## Conclusion
This project demonstrates how machine learning and data visualization can be used together to generate meaningful insights and support decision-making in real estate analytics.
